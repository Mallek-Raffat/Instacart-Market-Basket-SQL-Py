# 🛍️ Instacart Customer Segmentation

A complete end-to-end **Data Science** project that segments Instacart customers based on their purchasing behavior using **unsupervised machine learning**. The project combines **SQL** for data engineering, **Python** for exploratory data analysis, feature engineering, and **K-Means clustering** to identify distinct customer groups and generate actionable business insights.

---

# 📌 Project Objective

The objective of this project is to identify meaningful customer segments by analyzing purchasing behavior and shopping habits.

Instead of predicting a target variable, this project discovers hidden customer groups that can be used for:

- Personalized marketing
- Customer retention strategies
- Product recommendations
- Customer profiling
- Business decision-making

---

# 📂 Dataset

**Dataset Source**

https://www.kaggle.com/datasets/psparks/instacart-market-basket-analysis

The project uses the following tables:

- orders
- order_products_prior
- products
- aisles
- departments

Because the original dataset contains over **32 million rows**, a sampled dataset (~443K rows) was created for efficient local analysis.

## 5. Customer Segmentation

The project uses **K-Means Clustering** to segment customers.

### Why K-Means?

- Efficient for customer segmentation tasks
- Fast on medium-sized datasets
- Easy to interpret
- Widely used in marketing analytics
- Produces meaningful customer groups

The optimal number of clusters was determined using:

- Elbow Method
- Silhouette Score

The final model uses **K = 4** clusters to achieve a balance between cluster quality, interpretability, and business usefulness.
